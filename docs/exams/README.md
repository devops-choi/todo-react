## 📊 문제 유형별 설명

### 🔘 객관식 (5지선다)
- **배점**: 3-4점
- **특징**: 핵심 개념과 용어 이해도 평가
- **예시**: HTTP 메서드, React Hook, CSS 속성 등

### ✍️ 주관식 (단답형)
- **배점**: 4점
- **특징**: 정확한 용어나 코드 작성 능력 평가
- **예시**: 함수명, 파일 경로, 설정 값 등

### 📝 빈칸 채우기
- **배점**: 2-4점
- **특징**: 문맥 이해와 정확한 용어 사용 평가
- **예시**: CSS 속성명, JavaScript 키워드 등

### 📖 서술형
- **배점**: 14-18점
- **특징**: 개념 이해와 논리적 설명 능력 평가
- **예시**: 개념 설명, 장단점 비교, 과정 서술 등

### 💻 코딩 문제
- **배점**: 15-18점
- **특징**: 실제 코드 작성과 문제 해결 능력 평가
- **예시**: 컴포넌트 구현, API 함수 작성 등
